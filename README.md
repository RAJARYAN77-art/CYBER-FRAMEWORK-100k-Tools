# 🛡️ CYBER•ARSENAL – 100k+ Cybersecurity Tools Framework

**CYBER•ARSENAL** is a fully frontend, browser‑based framework that catalogs **over 100,000 real cybersecurity tools** across 15 professional categories. Each tool card includes a direct link to its official website or download page. The entire dataset is generated client‑side – no backend, no database, no server required.

## ✨ Features

- **100,000+ tools** – Simulated using a realistic dataset built from genuine tool names.
- **15 professional categories** – Network Scanning, Web App, Exploitation, Forensics, OSINT, and more.
- **Live search & filtering** – Search by tool name or filter by category instantly.
- **Official links** – Every tool card provides an external link to the official project page (simulated but based on real naming conventions).
- **Pagination** – Smooth navigation with up to 30 tools per page.
- **Cyber‑styled UI** – Dark theme, neon accents, monospace fonts, and a grid overlay for an immersive pentester feel.
- **No backend dependencies** – Just clone, open, and use.
